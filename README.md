# Amar Market

Amar Market is a simple, single-page web application for managing an online store. It provides basic e-commerce functionalities, including product management, customer registration, and order processing.

## Features

- **Admin Panel**: Manage products, customers, and orders.
- **Customer Panel**: Register, log in, and purchase products.
- **Product Management**: Add, edit, and delete products.
- **Shopping Cart**: Add products to a cart and proceed to checkout.
- **Order Tracking**: View order history and details.

## Getting Started

To run this project, simply open the `index.html` file in your web browser.

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox, Safari).

### Usage

1. **Admin**:
   - Open `index.html` and click the "Admin" button.
   - Use the default credentials: `admin` / `1234`.
   - From the dashboard, you can manage products, view customer information, and track orders.

2. **Customer**:
   - Open `index.html` and click the "Customer" button.
   - Register for a new account or log in with existing credentials.
   - Browse products, add them to your cart, and complete the checkout process.

## Project Structure

- `index.html`: The main HTML file containing the structure of the web page.
- `css/style.css`: The stylesheet for the application.
- `js/app.js`: The JavaScript file containing the application logic.
- `images/`: A directory for product images.
